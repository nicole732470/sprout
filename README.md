# Sprout 🌱

桌面悬浮小组件，用两盆会生长的植物追踪求职进度。

| 指标 | 周期 | 默认目标 |
|------|------|----------|
| LinkedIn Connect | 每周 | 100 |
| PM 岗位投递 | 每天 | 50 |

## 特点

- **植物生长**：点击花盆 +1，从种子 → 发芽 → 长叶 → 开花
- **智能提示**：根据剩余目标、时间进度、连续天数给出建议
- **本地运行**：数据存在 `data/progress.json`，可一键同步 GitHub
- **快捷键**：`⌘⇧C` Connect · `⌘⇧A` 投递

## 启动

```bash
npm install
npm start
```

拖顶部标题栏移动窗口。修改目标请编辑 `data/progress.json` 中的 `goals`。
